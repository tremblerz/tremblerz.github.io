Projects Done

Few of my projects

Security Analysis

Code analysis for exposing vulnerabilities of Web applications

This project is part of my Software Engineering course which I have extended as a individual project to do research in the field of code analysis techniques. The web application scanner is wriiten in Python with Django as framework. I have used VPS by Digital Ocean to host this project on web. Various production related skills such as hosting a secure server, adding SSL certificates for end to end encryption, continuous integration, project management and many more has been implemented in this project.

Vigilante-guide

Walking stick for blind

This project involved development of prototype as well as research. Demo paper for this project got accepted in ACM MobiHoc 2016. Our team made an Android application which was interfaced with bluetooth to communicate with walking stick. Research was done on orientation of Ultrasonic sensors and Bluetooth communication to minimize the delay.

Neural Cryptography

Neural Cryptography

This is my ongoing research work on Key exchange using neural networks. It is part of research project in the course of Principles of Information Security. Currently review paper on the various methods used before to do this is bieng written with more focus on cryptanalysis of different attacks such as genetic, probabilistic and geometric attacks.

Web scrapping

Scrapping web to identify radical networks and terrorist activities.

This project aims to identify radical networks by crawling over various social media and content writing platforms through Scrapy and Twitter APIs. Google search queries and twitter search queries is performed in this tool to gather the results. Once the results are retrieved data is put into PostgreSQL database through Scrapy pipe-lining afterwards sentiment analysis is performed over data which was gathered and filtered using textblob.

Operating Systems

Simiulation of various OS programs

This software was designed in many different different modules such as page replacement, disk handling, CPU schedular, dining-philosopher. This programs were written in Java and GUI was designed using swing library. This was done under the course of Operating systems to help us implement working of theoretical models and to give essence of multithreading combined with GUI

Music Mania

Music Mania

Music Mania is used for combining Video and Audio platforms by using their APIs. This project was done as part of ITWS-3 course to make us familiar with full stack development. Technologies used in this projects are Web2py as framework, Javascript and Bootstrap for interactive design.

Digital Equalizer

Speech signal enhancement using digital filters.

This project tries to enhance the speech signal and contrast the factor in filters like Frequency range, Attenuation and SNR for Digital filters. Implemented using MATLAB. To demonstrate working of above mentioned factors of an audio signal, an equalizer was developed on MATLAB with GUI.

Robot

Obstacle avoiding Robot using ultrasonic sensors

Developed a mini robot which can avoid the obstacle present on its path by taking most optimal path. This robot holds the capability to solve a simple maze by bruteforcing technique. This project was part of ITWS-2 course to provide hands on experience over hardware and software interfacing.
